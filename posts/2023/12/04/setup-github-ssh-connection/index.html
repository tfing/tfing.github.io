Setup Github Ssh Connection

Dec 4, 2023 · 115 words · 1 minute read

Setup the ssh connection of my laptop with Github by following the steps in the Github SSH tutorial.

Following is the record of my setup. It’s a simplified version of the Github tutorial.

  1. generate ssh key ssh-keygen -t ed25519 -C "tf1515@gmail.com" or ssh-keygen -t rsa -C "tf1515@gmail.com"

  2. reuse ssh key if use existing key, copy ssh private/public key files to ~/.ssh. modify the private key file permission because the ssh agent will check it. my private/public key file name is id_rsa_tinghan_github/id_rsa_tinghan_github_pub.

chmod 600 ~/.ssh/id_rsa_tinghan_github

  1. add public key to Github account https://docs.github.com/en/authentication/connecting-to-github-with-ssh/adding-a-new-ssh-key-to-your-github-account#adding-a-new-ssh-key-to-your-account

  2. add private key to ssh agent

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa_tinghan_github
  1. test ssh connection and done if see messages like below
ssh -T git@github.com
