Record info/links/commands used to setup this site and post articles.

offcial site: https://gohugo.io/getting-started/usage/ markdown syntax: https://www.markdownguide.org/tools/hugo/

Add post üîó

hugo new content/posts/<post-name>.md`

Insert <!--more--> in the post to identify the content showed as summary.

Add showToc: true in the head of post to add table of contents in the post.

pre-view drafts: hugo server -D

Deploy üîó

generate the public folder for deploy on Github. the HUGO_ENV=production is MUST to enable google analytics.

HUGO_ENV=production hugo --config config.yaml,config.toml

copy the generated public folder to the github page repo

cd mysite
rsync -ahv public/ ../tfing.github.io/ --delete --exclude=.git

After copy done, commit and push to github. check https://tfing.github.io for the result.

Update theme üîó

do hugo mod get -u

Install theme by Hugo module üîó

ref: https://scripter.co/hugo-modules-importing-a-theme/

  1. Init the site to be a hugo module and create the config.toml file
cd mysite
hugo mod init tfing.github.io/tfing
touch config.toml
  1. add these lines in config.toml
[module]
  [[module.imports]]
    path = "github.com/nodejh/hugo-theme-mini"
  1. do hugo server to auto download the theme

Update hugo binary üîó

  1. install git-lfs (do once)

    exec the script at https://packagecloud.io/github/git-lfs/install#bash-deb

    do sudo apt install git-lfs

    do git lfs install

  2. track the big size file in the repo

    git lfs track <file>
git add <file>
